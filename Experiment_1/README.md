Aim:-
To explore the Logisim software.

Theory:-
Logisim is a graphical simulation tool used for designing and analyzing digital logic circuits. It allows users to create circuits using basic components like logic gates, input/output devices, and wiring tools.

Digital logic gates are the fundamental building blocks of digital electronics. Each gate performs a specific logical operation based on binary inputs (0 and 1):

Logisim provides a visual interface where users can simulate these gates and observe real-time outputs using indicators like LEDs.

Procedure:-
1. Open Logisim software and create a new project.
2. From the toolbar, select input components (such as switches or constant inputs) and place them on the workspace.
3. Select the required logic gates (NOT, AND, OR, NAND, NOR, XOR, XNOR) from the gates library.
4. Connect the inputs to the respective gates using wiring tools.
5. Attach output devices such as LEDs to the output of each gate.
6. Set input values (0 or 1) and observe the corresponding output on the LEDs.
7. Repeat the process for different combinations of inputs to verify the behavior of each logic gate.

Result:-
The behavior of all basic logic gates was successfully observed using Logisim. The outputs obtained for different input combinations matched the expected truth tables of the respective gates.

Conclusion:-
The experiment provided a clear understanding of the working of basic digital logic gates through simulation. Logisim proved to be an effective and user-friendly tool for visualizing and analyzing digital circuits, reinforcing theoretical concepts through practical implementation.
